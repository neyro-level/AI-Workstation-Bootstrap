# AI Workstation Bootstrap

Публичная заготовка рабочего пространства для владельца бизнеса, менеджера проектов или специалиста, который хочет работать с AI-ассистентом системно.

Это не starter для сайта и не кодовый фреймворк. Это структура папок, документов, шаблонов и базовых Codex-настроек.

## Для кого

- предприниматель;
- менеджер проектов;
- маркетолог;
- консультант;
- руководитель команды;
- специалист, у которого есть основной бизнес и несколько рабочих проектов.

## Что разворачивается

```text
AI Workstation Bootstrap/
├── business-profile/     # кто вы, как работает бизнес, принципы, продукты
├── projects/             # папки рабочих проектов
├── system/               # системные настройки, серверы, интеграции, секреты
├── templates/            # шаблоны документов
├── design-base/          # база дизайн-референсов и дизайн-систем
├── codex-template/       # пример глобальных Codex-настроек и skills
├── docs/                 # инструкции по установке и работе
├── scripts/              # PowerShell-скрипты для установки/проверки
└── archive/              # архив старых проектов
```

## Главная идея

AI работает лучше, когда у него есть:

- профиль владельца / бизнеса;
- правила работы;
- структура проектов;
- паспорт каждого проекта;
- живой worklog;
- брифы страниц/документов/задач;
- отдельное безопасное хранилище секретов;
- понятные skills.

## Быстрый старт

1. Скопируйте эту папку в удобное место.
2. Заполните `business-profile/PROFILE.md`.
3. Для каждого проекта создайте папку в `projects/` по шаблону из `templates/project/`.
4. Установите Codex и скопируйте `codex-template/` в вашу глобальную папку Codex по инструкции `docs/03-codex-setup.md`.
5. Секреты храните только в Doppler или другом password/secrets manager.
6. Если планируете работать с дизайном внутри VS Code, настройте Pencil по `docs/08-pencil-setup.md`.
7. Для установки Pencil OS skills в Codex используйте `docs/09-pencil-os-skills.md` или `scripts/install-pencil-skills.ps1`.

## Codex skills

В `codex-template/skills/` лежат базовые skills для рабочей станции:

- `skill-creator` — создание и обновление Codex skills по официальной структуре OpenAI;
- Pencil OS skills — набор навыков для работы с Pencil.dev.

## Pencil OS

В комплект входит мини-операционная система skills для Pencil.dev:

- `pencil-os` — главный роутер Pencil-задач;
- `pencil-mcp-operator` — безопасная работа с `.pen` через Pencil MCP;
- `pencil-modern-ui` — современный UI, shadcn/Radix, Heroicons, anti-AI-slop;
- `pencil-b2b-sites` — B2B-сайты, Hero, CTA, формы, каталоги;
- `pencil-design-system` — tokens, variables, lint, export/handoff.

Установка описана в `docs/09-pencil-os-skills.md`.

## Безопасность

В этом репозитории не должно быть реальных API-ключей, токенов, паролей, приватных SSH-ключей, `.env` и клиентских секретов.

Допустимы только:

- `.env.example`;
- `config.example`;
- инструкции;
- шаблоны.
