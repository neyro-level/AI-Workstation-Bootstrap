# MCP Setup

MCP-серверы подключаются опционально. Каждый пользователь регистрируется в сервисах сам и хранит ключи в Doppler.

## Базовый список

- Playwright MCP — браузерная автоматизация.
- Firecrawl MCP — чтение сайтов.
- Tavily MCP — AI-поиск.
- YouTube Transcript MCP — транскрипты видео.
- GitHub MCP — репозитории и issues.
- Google Drive / Docs — документы и таблицы.
- Figma — дизайн.
- Pencil MCP — `.pen` дизайн-файлы внутри VS Code/Pencil.

## Общий порядок

1. Зарегистрироваться в сервисе.
2. Получить API key, если нужен.
3. Положить ключ в Doppler.
4. Подключить MCP по инструкции клиента.
5. Проверить список подключений.

Не копируйте чужие MCP-конфиги с реальными ключами.

Для Pencil используйте отдельный чек-лист: `docs/08-pencil-setup.md`.
