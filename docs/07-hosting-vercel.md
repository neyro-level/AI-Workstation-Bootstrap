# Hosting: Vercel

Vercel — простой рекомендуемый вариант для статических сайтов, frontend-проектов и прототипов.

## Что нужно

1. GitHub account.
2. Vercel account.
3. Vercel CLI.
4. Репозиторий проекта.
5. Environment variables в Vercel или Doppler integration.

Vercel не обязателен для всех проектов. Это рекомендация для простого старта.
