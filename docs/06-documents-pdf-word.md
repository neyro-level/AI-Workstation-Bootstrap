# Documents, Word and PDF

Для работы с документами рекомендуется:

- VS Code Markdown extensions;
- Python;
- `python-docx`;
- `pypdf`;
- `pdfplumber`;
- `pandas`;
- `openpyxl`;
- LibreOffice CLI;
- Pandoc.

## Правило хранения

- рабочие документы бизнеса → `business-profile/` или `projects/`;
- клиентские/проектные документы → папка проекта;
- шаблоны → `templates/`;
- старое → `archive/`.

Не смешивайте документы с секретами.
