# Design Base

Папка для базы дизайнов, дизайн-систем и визуальных референсов.

Можно хранить:

- скриншоты;
- ссылки на Figma;
- PDF/презентации;
- дизайн-системы;
- подборки интерфейсов;
- заметки по паттернам.

Рекомендуемая структура:

```text
design-base/
├── 00_INDEX.md
├── b2b/
├── ecommerce/
├── dashboards/
├── real-estate/
├── services/
└── references/
```

Не храните здесь чужие приватные материалы без права использования.
