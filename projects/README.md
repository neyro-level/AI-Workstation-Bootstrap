# Projects

Здесь создаются папки рабочих проектов.

Проект — это не обязательно клиент. Это может быть:

- внутренний продукт;
- запуск направления;
- исследование;
- сайт;
- операционная задача;
- интеграция;
- рекламная кампания;
- личный рабочий проект.

## Рекомендуемая структура проекта

```text
project-name/
├── PASSPORT_PROJECTS.md
├── WORKLOG.md
├── docs/
├── pages/
├── assets/
└── archive/
```

Для создания нового проекта используйте шаблоны из `templates/project/`.
