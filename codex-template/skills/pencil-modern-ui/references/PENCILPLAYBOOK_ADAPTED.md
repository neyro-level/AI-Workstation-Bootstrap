# PencilPlaybook Adapted

Источник идей: https://github.com/stevembarclay/pencilplaybook

Использовать как адаптированный набор принципов, не копировать дословно.

## Canvas Archaeology

Перед правками:

1. Прочитать активный файл.
2. Понять top-level frames.
3. Найти selection или нужную область.
4. Проверить variables.
5. Только потом менять.

## Skeleton First

Сначала:

- frame;
- сетка;
- основные зоны;
- навигация;
- CTA;
- контентные блоки.

Потом:

- текст;
- компоненты;
- изображения;
- visual polish.

## Perceptual Defaults

- Disabled opacity около 40%.
- Hover должен быть заметен, но не кричать.
- Body text на темном фоне не делать чисто белым без причины.
- Body line-height около 1.5.
- Heading line-height около 1.1-1.2.
- Touch target на mobile минимум 44px.
- Primary button высотой около 40-44px.
- Compact button около 32px.
- Focus ring должен быть видимым.

## Canvas Hygiene

- Не складывать все экраны в один хаотичный frame.
- Называть frames по смыслу.
- Не плодить случайные one-off styles.
- Не делать больше крупной партии операций без проверки layout.
